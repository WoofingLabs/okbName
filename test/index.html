How It Works?

Mint your unique domain name NFT on X Layer with our Launchpad! It’s simple and fast:

  • Connect Your Wallet: Use MetaMask to connect to X Layer (chain ID: 196). Ensure you have enough OKB for .okb or ETH for .star, .xdog, .okx, .dao, .ai, .x gas fees.
  • Select Domain Type: Choose .okb, .star, .xdog, .okx, .dao, .ai, or .x using the toggle buttons.
  • Mint Your Domain: Enter a domain ID (1-15 alphanumeric characters, e.g., “jace123”). The NFT will be minted with the selected suffix.
  • Manual Registration: Copy the hex data and contract address to register manually via a wallet transaction with 0 ETH/OKB and the hex data as msg.data.
  • View Your Domains: Fetch your minted domain NFTs, including token IDs and full domain names (e.g., jace123.okb).
  • Secure & Unique: Each address can mint only one domain per contract, and each ID is unique (case-insensitive). Built on X Layer for low fees and secure NFT creation.
